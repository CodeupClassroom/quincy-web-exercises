Relative Path

  • Relative paths (any path that does not start with /
  • Starts looking in the current folder of this particular HTML page..
  • .. means go up one folder
  • ../.. means go up two folders
  • ../img/arrow.png means go up one folder, then into img folder, then find arrow.png

Absolute Paths

Absolute paths in HTML start with the forward slash character.

/ means "root" and for web apps/pages, root means public folder on your server
